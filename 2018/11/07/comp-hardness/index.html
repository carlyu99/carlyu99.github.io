P、NP、co-NP问题

Posted on 2018-11-07 | | views

是非题：任一输入对应的输出只能是 Y 或 N 的问题。

对于一个是非题，称答案是 Y 的输入为 Y 输入，答案为 N 的输入为 N 输入。

P 问题：存在多项式算法，对 Y 输入输出 Y，对 N 输入输出 N。

NP 问题：存在多项式算法，对于 Y 输入，存在一个附加信息使得算法输出 Y；对于 N 输入，不存在一个附加信息使得算法输出 Y。

举例：判断一个逻辑式是否可满足。存在多项式算法来验证一种赋值方案能否使逻辑式为真。对于可满足的逻辑式（Y 输入），存在一种这样的赋值方案作为附加信息。对于不可满足的逻辑式（N 输入），不存在这样的信息。

co-NP 问题：存在多项式算法，对于 N 输入，存在一个附加信息使得算法输出 N；对于 Y 输入，不存在一个附加信息使得算法输出 N。

举例：判断一个逻辑式是否 不 可满足。存在多项式算法来验证一种赋值方案能否使逻辑式为真。对于可满足的逻辑式（N 输入），存在一种这样的赋值方案作为附加信息。对于不可满足的逻辑式（Y 输入），不存在这样的信息。

It is believed that NP ≠ co-NP.
